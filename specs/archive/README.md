# archive/

Frozen historical material. **Do not edit files in this tree.**

Everything under `archive/` is preserved verbatim from the pre-rename project or from early iterations of `lazy-harness`. The value of this tree is provenance — it records how the framework got to its current shape. Editing a file here to fix a stale path, a reference to an old name, or a broken link defeats the purpose.

If you find content here that contradicts the current code or docs, treat it as history, not truth. The current state of the repo is the source of truth.

## Contents

- **`adrs-legacy/`** — ADRs 001–013 from the predecessor project. Kept for decision provenance. Originally lived at `docs/architecture/decisions/legacy/`.
- **`history/`** — Early `lazy-harness` plans, specs, workflows, genesis notes, and lessons learned. Originally lived at `docs/history/`.
